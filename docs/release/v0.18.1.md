# Release v0.18.1

**Release Date:** 2026-03-16

## Highlights

The Iconik SOW generator now prompts the user for a Google Drive folder link during the interview, rather than silently using a hardcoded default destination.

## Changes

- **`/sow-iconik` destination folder** — The interview now asks "Where should I save the finished SOW?" and accepts a Google Drive folder link. The default folder is still available as a fallback when the user skips or says "default."

## Upgrade Instructions

No action needed. The skill will prompt for a folder link on next use.
