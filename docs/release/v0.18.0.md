# Release v0.18.0

**Release Date:** 2026-03-15

## Highlights

Trellis expands beyond development workflow into a comprehensive AI-native framework covering client deliverables, documentation, and domain knowledge. This release adds 5 new SOW generator skills (with Iconik fully built out and 3 placeholders), a repo documentation generator, and reframes the project identity accordingly.

## What's New

- **`/sow-iconik`** — Full Iconik Up and Running SOW generator with template-driven interview, tag-based conditional logic, context intake, and logo insertion. Reads variables and options from the template at runtime — no hardcoded lists.
- **`/sow-catdv`** — CatDV Up and Running SOW placeholder (template exists, interview not yet built)
- **`/sow-catdv-upgrade`** — CatDV Upgrade & Cloud Migration SOW placeholder
- **`/sow-dhub`** — Dhub OTT SOW placeholder
- **`/trellis:repo-docs-generator`** — Generates a three-document package (Technical Overview, Deployment & Troubleshooting Guide, User Guide) from any codebase via turn-based interview

## Changes

- `/sow` renamed to `/sow-migration` — migration SOW generator now scoped to MAM and Avid Interplay migrations only
- Migration SOW generator gains Context Intake (Step 0) for pre-filled interviews from BOMs, emails, or notes
- Iconik SOW removed marvin compatibility metadata (targets Claude Code and Gemini gems)
- README, CLAUDE.md, and plugin manifest updated to reflect Trellis as an AI-native workflow framework

## Breaking Changes

- `/sow` slash command no longer exists — use `/sow-migration` instead

## Upgrade Instructions

Update your plugin installation. If you had scripts or aliases referencing `/sow`, update them to `/sow-migration`.
