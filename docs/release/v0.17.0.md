# Release v0.17.0

**Release Date:** 2026-03-06

## Highlights

The SOW generator skill has been renamed to `migration-sow-generator` to clearly scope it to Migration SOWs (MAM and Avid Interplay). Legacy python-docx scripts have been removed — all generation now happens via Google Docs API.

## What's Changed

- **Renamed skill**: `sow-generator` → `migration-sow-generator` — use `/trellis:migration-sow-generator` going forward
- **Removed legacy scripts**: `scripts/generate_sow.py`, `scripts/__init__.py`, `scripts/requirements.txt` deleted — the Google Docs API-native workflow is the only path now

## Upgrade Instructions

- Replace any references to `/trellis:sow-generator` with `/trellis:migration-sow-generator`
- No functional changes to the generation workflow itself
