# Release v0.5.1

**Release Date:** 2026-01-15

## Highlights

This release adds comprehensive documentation with a step-by-step workflow guide (USAGE.md) that helps users understand the full Trellis workflow from PRD creation through implementation.

## What's New

- **USAGE.md** - Complete workflow guide covering:
  - PRD creation with `/trellis.prd`
  - Epic decomposition with `/trellis.epics`
  - Specification authoring with `/speckit.specify`
  - Implementation execution with `/trellis.implement`
  - Detailed command references and practical examples
  - Troubleshooting tips and best practices

None in this release.

None in this release.
