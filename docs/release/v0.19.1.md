# Release v0.19.1

**Release Date:** 2026-03-16

## Highlights

The Iconik SOW generator now collects a client code during the interview, prefixing it to the generated document name for easier identification and filing.

## What's New

- Added `#client-code` global variable to the Iconik SOW generator interview (asked first, before client name)
- Document naming updated to `{client-code} {client} Iconik Up and Running SOW - {date}`
