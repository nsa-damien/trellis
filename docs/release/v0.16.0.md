# Release v0.16.0

**Release Date:** 2026-03-03

## Highlights

New project-requirements skill for generating client-facing requirements documents through a guided interview. This fills a key gap in the client engagement workflow — requirements are captured and approved before investing in SOW and pricing.

## What's New

- **`/trellis:project-requirements`** — Generate professional, client-facing project requirements documents through a guided interview. Captures project background, deliverables, assumptions, exclusions, dependencies, success criteria, and timeline. Produces structured markdown for client sign-off before the SOW/pricing phase.

  **Workflow position:** Requirements Doc → Client Approval → SOW → Quote

## Upgrade Instructions

Standard plugin update — no breaking changes from v0.15.1.
