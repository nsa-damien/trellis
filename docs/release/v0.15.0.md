# Release v0.15.0

**Release Date:** 2026-03-03

## Highlights

Two new productivity skills join the Trellis toolkit: SOW generation from Google Docs templates and meeting summary email creation with Google Calendar integration. The SOW generator also received a major architectural upgrade, moving from local python-docx processing to the Google Docs API.

## What's New

- **`/trellis:sow-generator`** — Generate client Statements of Work from Google Docs templates via a guided interview. Interviews for project details (client, scope, timeline, pricing), duplicates the template in Google Drive, fills placeholders and applies conditional logic via the Google Docs API. Supports MAM Migration and Avid Interplay Migration templates.

- **`/trellis:meeting-summary`** — Generate professional meeting summary emails from a guided interview. Pulls participant info from Google Calendar, accepts notes and transcripts inline or via Google Drive links, and produces a ready-to-send summary email with optional Gmail draft creation.

## Changes

- **SOW generator migrated to Google Docs API** — Template duplication and placeholder replacement now happen directly in Google Drive, removing the python-docx dependency and delivering a Google Doc link instead of a local .docx file.

## Upgrade Instructions

Standard plugin update — no breaking changes from v0.14.0.
